# Consistent Hash

This directory contains implementation of consistent hash and (simulated, not really distributed) distributed key-value store 
using the hash.  

## Build
```
cmake -S . -B build
cmake --build build
```

## Run `main.cc`

```
./build/src/main
```

## Run `consistent_hash_test.cc`

```
./build/lib/consistent_hash_test
```
